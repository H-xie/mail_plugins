# Thunderbird 插件：作业邮件联系人采集

这个插件会在你手动右键执行时：

- 读取发件人的邮箱地址
- 从邮件主题提取姓名
- 从邮件主题提取学号
- 从邮件主题提取课程
- 将信息保存到名为“作业课程学生”的通讯录
- 不会自动处理新来信

## 1. 主题格式建议

推荐使用以下任一格式，插件默认可识别：

1) 带标签（推荐）

姓名: 张三 学号: 20230001 课程: 编译原理

2) 简化格式

作业-张三-20230001-编译原理

3) 你新增的格式（已支持）

20230001+张三+编译原理+第一次作业

如果你的主题格式不同，请修改 `background.js` 中的正则表达式：

- `nameRegex`
- `studentIdRegex`
- `courseRegex`
- `subjectFallbackRegex`

## 2. 安装与调试（临时加载）

1. 打开 Thunderbird
2. 进入 工具 -> 开发者工具 -> 调试附加组件
3. 点击 “临时载入附加组件”
4. 选择本目录下的 `manifest.json`
5. 在邮件列表中选中一封或多封邮件
6. 右键 -> 保存选中邮件到课程联系人
7. 检查通讯录是否新增或更新联系人

## 3. 联系人保存规则

- 以发件邮箱（PrimaryEmail）作为唯一键
- 若邮箱已存在：更新显示名和备注
- 若邮箱不存在：创建新联系人
- 备注字段中包含：学号、课程、来源主题

## 4. 注意事项

- 临时加载的插件在 Thunderbird 重启后会失效，需要重新加载
- 若你希望正式分发，需要打包并签名扩展
- 当前逻辑仅在右键菜单手动触发，避免误采集
